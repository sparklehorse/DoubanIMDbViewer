#DoubanIMDbViewer
本扩展可以在豆瓣电影页面查看IMDB以及烂番茄评分。


##Changes
* v1.6 修正标题没有年份时烂番茄显示问题。新增一个在imdb官网的评分查询，以解决原api查询不正常的问题。
* V1.8 增加评分载入过程中的动画

##Thanks
评分查询使用的是seansay所写的API，详见[https://github.com/seansay](https://github.com/seansay "https://github.com/seansay") ，在此向seansay表示感谢。
